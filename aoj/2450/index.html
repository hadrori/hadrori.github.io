AOJ 2450 Do use segment tree

Category: AOJ Tag: heavy-light-decomposition, segment-tree

Do use segment tree

問題概要

各頂点に重さを持つn(<=200,000)頂点の木が与えられる.
以下のクエリが合計でq(<=100,000)個くるのでその中の出力クエリに答えろ.

  • 頂点aからbへのパス上の頂点の値をcに変更する
  • 頂点aからbへのパス上の連続する頂点の和の最大

解法

木をHL分解してセグ木で殴るだけ
